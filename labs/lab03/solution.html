Lab No. 03: Solution¶

Requirements¶

Requirements for generate_boards¶

  1. GB1: The function shall take no arguments and return a 3 element tuple
  2. GB2: The first element of the returned tuple shall be a two dimensional array of size 9 x 9 with exactly 10 elements determined at random with integer values equal to -1 and the remaining elements with integer values between 0 and 8, which will depend on the number of neighboring cells (including diagonals) that have a -1 value.
  3. GB3: The second element of the returned tuple shall be an array of size of 9 x 9 and all of its elements shall contain a single empty space string.
  4. GB4: The third element of the returned tuple shall be a list of the indices of the elements that have a value equal to -1 in the first returned array, and as such it shall contain exactly ten distinct elements. The indices shall be represented as a two element tuple where the first element corresponds to the row index and the second element to the column index

Requirements for update_neighbors¶

  1. UN1: The function shall take as input a two dimensional array of integers of size 9 x 9 and the row and column indices of an element and shall update the input array in a way such that the cells that are adjacent vertically, horizontally and diagonally have their values increased by 1, with the exception of those that have a value equal to -1 which shall remain unmodified.

Requirements for open_squares¶

  1. OS1: The function shall take as input the row and column indices of an element of a two dimensional 9 x 9 array, a two dimensional array of strings of size 9 x 9, and a two dimensional array of integers of size 9 x 9. The function shall update the third input array’s element that corresponds to the coordinates given by the first two imputs with the corresponding string representation found in the fourth input array, if it’s value is not equal to 0, otherwise it should update all the horizontally, vertically and diagnonal neighboring elements as well. Neighboring elements that have values equal to 0 shall also have their neighbors updated, and any additional elements that have a value of 0 shall have their values updated as well.

Tests for update_neighbors¶

For the update_neighbors function tests we will refer to the input values as board, row and column. The last two when referenced in conjuction will be refered as the input coordinates The behavior of this function is affected by:

  1. The input coordinates and their position relative to the edges of the board: left, right, middle, top and bottom
  2. The presence of already updated elements in the vicinity of the input coordinates
  3. The presence of bombs in the vicinity of the input coordinates

Unit Tests¶

"""
Tests for Lab 03.

Note:
As it will be seen in class, many of these tests are unreliable
and will pass/fail in an unpredicatable manner since boards are
generated at random and the tests do not exercise any mechanism to
enforce test repeatability.
This tests also do not use fixtures efficiently.
"""

import unittest
from minesweeper import generate_boards, update_neighbors, open_squares

class GenerateBoardsTest(unittest.TestCase):


    def test_1_output_type_validation(self):
        """
        The output data structures have the expected sizes
        """
        board, opened_cells, bomb_list = generate_boards()
        self.assertEqual(9, len(board))
        for row in board:
            self.assertEqual(9, len(row))

        self.assertEqual(9, len(opened_cells))
        for row in opened_cells:
            self.assertEqual(9, len(row))

        self.assertEqual(10, len(bomb_list))
        for element in bomb_list:
            self.assertEqual(2, len(element))


    def test_2_board_validation(self):
        """
        The generated board has the expected number of bombs
        """

        board, _, _ = generate_boards()
        bomb_count = 0
        for row in board:
            for cell in row:
                if cell == -1:
                    bomb_count += 1
        self.assertEqual(10, bomb_count)


    def test_3_opened_cells_validation(self):
        """
        The generated opened_cells are all in unopened state
        """

        _, opened_cells, _ = generate_boards()
        for row in opened_cells:
            for cell in row:
                self.assertEqual(' ', cell)

    def test_4_bombs_coord_validation(self):
        """
        The bomb coordinates match an element whose value is -1 in the board
        """

        board, _, bombs = generate_boards()
        for row, col in bombs:
            self.assertEqual(-1, board[row][col])

    def test_5_bombs_coord_unique(self):
        """
        All the elmements in the generated bomb list are unique
        """
        _, _, bombs = generate_boards()
        bombs_as_set = set()
        for item in bombs:
            bombs_as_set.add(item)
        self.assertEqual(10, len(bombs_as_set))

    def test_6_boards_random(self):
        """
        boards are generated randomly
        """
        board1 = generate_boards()
        board2 = generate_boards()
        self.assertNotEqual(board1, board2)

    def test_7_board_neighbor_calculation(self):
        """
        board cell values correspond to the count of surrounding bombs
        """
        board, _, _ = generate_boards()

        for row in range(9):
            for col in range(9):
                if board[row][col] != -1:
                    
                    bomb_count = 0
                    
                    top = max(0, row-1)
                    bottom = min(8, row+1)
                    left = max(0, col-1)
                    right = min(8, col+1)
                    
                    for neigh_row in range (top, bottom+1):
                        for neigh_col in range(left, right+1):
                            if board[neigh_row][neigh_col] == -1:
                                bomb_count += 1
                    self.assertEqual(board[row][col], bomb_count)


class UpdateNeighborsTests(unittest.TestCase):

    def test_1_empty_board(self):
        """
        Test empty board update on edges and center
        """
        coordinate_list = [(0,0), (0,4), (0,8), (4,0), (4,4), (4,8), (8,0), (8,4), (8,8)]
        for row, col in coordinate_list:
            board = [[0 for _ in range(9)] for _ in range(9)]
            update_neighbors(board, row, col)
            
            top = max(0, row-1)
            bottom = min(8, row+1)
            left = max(0, col-1)
            right = min(8, col+1)
           
            for neigh_row in range (top, bottom+1):
                for neigh_col in range(left, right+1):
                    if neigh_row != row and neigh_col != col:
                        self.assertEqual(1, board[row][col])


    def test_2_nonempty_board(self):
        """
        Test non empty board update on edges and center
        """
        coordinate_list = [(0,0), (0,4), (0,8), (4,0), (4,4), (4,8), (8,0), (8,4), (8,8)]
        for row, col in coordinate_list:
            board = [[1 for _ in range(9)] for _ in range(9)]
            update_neighbors(board, row, col)
            
            top = max(0, row-1)
            bottom = min(8, row+1)
            left = max(0, col-1)
            right = min(8, col+1)
           
            for neigh_row in range (top, bottom+1):
                for neigh_col in range(left, right+1):
                    if neigh_row != row and neigh_col != col:
                        self.assertEqual(2, board[row][col])


    def test_3_cell_with_neigboring_bombs(self):
        """
        Test empty board update on edges and center
        """
        coordinate_list = [(0,0), (0,4), (0,8), (4,0), (4,4), (4,8), (8,0), (8,4), (8,8)]
        for row, col in coordinate_list:
            board = [[-1 if (x + y) % 2 else 1 for y in range(9) ] for x in range(9)]
            update_neighbors(board, row, col)
            top = max(0, row-1)
            bottom = min(8, row+1)
            left = max(0, col-1)
            right = min(8, col+1)
           
            for neigh_row in range (top, bottom+1):
                for neigh_col in range(left, right+1):
                    if (neigh_row + neigh_col) % 2:
                        self.assertEqual(-1, board[neigh_row][neigh_col])
                    else:
                        self.assertEqual(2, board[neigh_row][neigh_col])


class OpenSquaresTests(unittest.TestCase):

    def test_1_neighboring_bombs(self):
        """
        Test open squares on cell with neighboring bombs
        """
        coordinate_list = [(0,0), (0,4), (0,8), (4,0), (4,4), (4,8), (8,0), (8,4), (8,8)]
        for row, col in coordinate_list:
            opened = [[' ' for _ in range(9)] for _ in range(9)]
            board = [[0 for _ in range(9)] for _ in range(9)]
            board[row][col] = 1
            open_squares(row, col, opened, board)
            self.assertEqual('1', opened[row][col])

    def test_2_no_neighboring_bombs(self):
        """
        Test open squares on cell with no neighboring bombs
        """
        coordinate_list = [(0,0), (0,4), (0,8), (4,0), (4,4), (4,8), (8,0), (8,4), (8,8)]
        for row, col in coordinate_list:
            opened = [[' ' for _ in range(9)] for _ in range(9)]
            board = [[1 for _ in range(9)] for _ in range(9)]
            board[row][col] = 0
            open_squares(row, col, opened, board)
            self.assertEqual('0', opened[row][col])

            top = max(0, row-1)
            bottom = min(8, row+1)
            left = max(0, col-1)
            right = min(8, col+1)
           
            for neigh_row in range (top, bottom+1):
                for neigh_col in range(left, right+1):
                    if neigh_row != row or neigh_col != col:
                        self.assertEqual('1', opened[neigh_row][neigh_col])
 
    def test_3_neighbors_with_no_bombs(self):
        """
        Test open squares where neighbors have no neighboring bombs
        """
        coordinate_list = [(0,0), (0,4), (0,8), (4,0), (4,4), (4,8), (8,0), (8,4), (8,8)]
        for row, col in coordinate_list:
            opened = [[' ' for _ in range(9)] for _ in range(9)]
            board = [[1 for _ in range(9)] for _ in range(9)]
            
            top = max(0, row-1)
            bottom = min(8, row+1)
            left = max(0, col-1)
            right = min(8, col+1)
           
            for neigh_row in range (top, bottom+1):
                for neigh_col in range(left, right+1):
                    board[neigh_row][neigh_col] = 0
            open_squares(row, col, opened, board)
           
            # check neighbors
            for neigh_row in range (top, bottom+1):
                for neigh_col in range(left, right+1):
                    if neigh_row != row or neigh_col != col:
                        self.assertEqual('0', opened[neigh_row][neigh_col])

            # check elements two cells away
            top = max(0, row-2)
            bottom = min(8, row+2)
            left = max(0, col-2)
            right = min(8, col+2)

            # top:
            if row-2 >= 0:
                for neigh_col in range (left, right):
                    self.assertEqual('1', opened[top][neigh_col])
           
            # bottom:
            if row+2 <= 8:
                for neigh_col in range (left, right):
                    self.assertEqual('1', opened[bottom][neigh_col])
            
            # left:
            if col-2 >= 0:
                for neigh_row in range (top, bottom):
                    self.assertEqual('1', opened[neigh_row][left])

            # right:
            if col+2 <= 8:
                for neigh_row in range (top, bottom):
                    self.assertEqual('1', opened[neigh_row][right])
